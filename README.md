# Quienesquien
# ¿Quién es Quién?

El clásico juego de mesa contra la inteligencia artificial.

## Cómo jugar

1. La app elige un personaje secreto para ti y otro para la IA
2. Ves tu tablero con las 24 caras
3. Elige una pregunta del desplegable y pulsa **Preguntar**
4. Toca las caras para eliminarlas del tablero
5. Cuando estés seguro, pulsa **Adivinar**

## Personajes

Los 24 personajes clásicos: Alex, Alfred, Anita, Bernard, Bill, Claire, David, Eric, Felipe, Franz, George, Herman, José, Maria, Max, Mia, Miguel, Molly, Nuria, Paul, Pedro, Philip, Richard y Susan.

## Tecnología

PWA pura — HTML, CSS y JavaScript. Sin dependencias externas. Funciona offline.
